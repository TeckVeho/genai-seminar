## __1\. 事業概要 \(Concept\)__

- __研修名:__ 物流企業特化 AI人材育成プログラム「Claude AI活用・業務変革コース（全10回）」
- __形式:__ パッケージ型研修プログラム（標準カリキュラム）
- __目的:  
__物流業界に共通する事務課題（電話対応、伝票入力、請求処理、社内文書作成等）を解決するためのAI活用スキルを習得する。受講者自身が業務を棚卸しし、**Claude Cowork** を活用して最適なソリューションを設計・構築することで、研修後すぐに活かせる実践力を身につける。
- __特徴:__
	- __業務課題起点の設計:__ 初回に自身の業務棚卸しを行い、座学でAIツールを学び、ワークショップで自分の課題を解決するソリューションを構築する一気通貫の構成。学びが「自分ごと」になる。
	- __Cowork によるエージェント型タスク実行:__ Claude Desktop の Cowork 機能を使い、複雑な業務タスク（文書生成・Excel作成・データ処理・ファイル整理等）を自然言語の指示だけで自動実行する。ツール選定の迷いがなく、全員が同一の操作体系で学べる。
	- __Claude プラットフォーム活用:__ Claude の基本機能（テキスト生成・Web検索・ファイル入力・Artifacts等）をベースに、Cowork（エージェント型タスク実行）まで1つのデスクトップアプリで完結。ローカルファイルに直接アクセスできるため、ファイルのアップロード・ダウンロード作業が不要で導入ハードルが低い。
	- __実務直結:__ 参加者が自身の業務課題を持ち込み、研修内で解決策を構築する（教材は共通だが、アウトプットは個別化される）。成果物はローカルの Excel・Word・PowerPoint 等としてそのまま業務に持ち帰れる。

## __2\. 対象企業・受講者 \(Target\)__

- __ターゲット:__ 物流業、運送業、倉庫業（中小企業）
- __対象職種:__ 一般事務（総務・経理・庶務等）
- __受講推奨人数:__ 1開催あたり 最大10名まで（少人数制）
- __前提スキル:__ 不問（IT初心者対応）

## __3-1\. 推奨環境・前提ライセンス一覧__

研修で扱う機能により、利用できる範囲が異なります。契約前に受講企業の環境と照合してください。

| 区分 | 内容 | 主に該当する回 |
| --- | --- | --- |
| __必須__ | PC（**macOS または Windows**）、インターネット接続、**Claude Desktop アプリ**（[claude.com/download](https://claude.com/download)）、Claudeアカウント（無料可 — Chat 機能のみ利用可） | 全回 |
| __推奨__ | **Claude Pro**（月額 $20）— Cowork 利用可。タスク実行の使用量枠が確保され、研修中の制限切れリスクを低減できる | 第03回〜 |
| __推奨__ | **Claude Team / Enterprise**（企業契約）— 上記に加え、管理者によるアクセス権限管理・メンバー管理が可能。Cowork のプロジェクト機能でチーム内のナレッジ共有にも活用できる | 第03回〜、第06〜09回 |

__フォールバック:__ 受講者のプラン状況は第01回の事前アンケートで確認し、第03回以降の進行に反映します。Pro 以上の契約がない場合は、講師アカウントで Cowork を画面共有し**デモ＋ペア作業**で体験させます。Chat モード（無料プランで利用可）での基本操作は全員が実施可能です。企業向け契約の有無で **入力してよいデータの範囲** が変わるため、[AI入力OK_NGチェックリスト.md](AI入力OK_NGチェックリスト.md) と社内ポリシーを必ず確認してください。なお、Cowork はローカル環境で実行されモデル学習には使用されませんが、ネットワークアクセスが発生する点、および Research Preview のため **監査ログ・コンプライアンス API の対象外** である点を事前に周知してください。
